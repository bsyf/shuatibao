# 刷题宝（框架）

一个手机端刷题学习的 PWA 框架。**本仓库只含界面与功能，不含任何题目**。

题库由使用者在自己的设备上导入（一个 `.json` 文件），只存在本地浏览器中，不会上传。

## 使用
1. 打开网址，安卓 Chrome / iPhone Safari「添加到主屏幕」即可像 App 一样使用。
2. 首次打开点「📥 选择题库文件导入」，选择自己的题库 `.json`。
3. 之后离线也能刷，答题进度自动保存在本机。

## 题库文件格式
```json
{ "PASSAGES": { "p1": { "text": "...", "translation": "..." } },
  "DATA": { "科目名": { "types": { "题型名": [ { "id":"...", "question":"...", "answer":"...", "explanation":"..." } ] } } } }
```
